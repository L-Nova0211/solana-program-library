# TypeScript bindings for stake-pool program

For use with both node.js and in-browser.

## Installation

```
npm install
```

## Build and run

In the `js` folder:

```
npm run compile
npm run lint
node dist/index.js
```

## Test

```
npm run compile
npm test
```

Sample output:

```
> stake-pool-js@0.0.1 test
```
